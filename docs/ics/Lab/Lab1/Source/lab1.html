指令

LC-3（Little Computer 3）是一种教学用的计算机体系结构，它的指令集设计得相对简单，易于理解。下面是一些基本的 LC-3 指令：

  1. ADD：这是一个加法指令，用于将两个数相加并将结果存储在一个寄存器中。格式为0001 DR SR1 0 00 SR2或0001 DR SR1 1 imm5，其中 DR 是目标寄存器，SR1 和 SR2 是源寄存器，imm5 是立即数。

    ADD   R2,R3,R4        ; R2  ← R3 + R4
ADD   R2,R3,#7        ; R2  ← R3 + 7

  2. AND：这是一个逻辑与指令，用于执行逻辑与操作并将结果存储在一个寄存器中。格式为0101 DR SR1 0 00 SR2或0101 DR SR1 1 imm5。

  3. BR：这是一个分支指令，用于根据条件标志（N，Z，P）改变程序的执行流。格式为0000 nzp PCoffset9，其中 nzp 表示负、零、正标志，PCoffset9 是 PC 相对偏移。

  4. JMP：这是一个跳转指令，用于无条件地改变程序的执行流。格式为1100 000 SR 000000，其中 SR 是源寄存器。

  5. JSR：这是一个跳转到子程序的指令。格式为0100 1 PCoffset11。

  6. JSRR：这是另一种跳转到子程序的指令。格式为0100 000 SR 000000。

  7. LD：这是一个加载指令，用于将内存中的值加载到寄存器中。格式为0010 DR PCoffset9。

  8. LDI：这是一个间接加载指令，用于通过间接地址将内存中的值加载到寄存器中。格式为1010 DR PCoffset9。

  9. LDR：这是一个基址+偏移量加载指令，用于将基址+偏移量地址处的内存值加载到寄存器中。格式为0110 DR BaseR offset6。

  10. LEA：这是一个加载有效地址指令，用于将有效地址加载到寄存器中。格式为1110 DR PCoffset9。

  11. NOT：这是一个逻辑非指令，用于对一个数进行逻辑非操作并将结果存储在一个寄存器中。格式为1001 DR SR 111111.

  12. ST：这是一个存储指令，用于将寄存器中的值存储到内存中。格式为0011 SR PCoffset9.

  13. STI：这是一个间接存储指令，用于通过间接地址将寄存器中的值存储到内存中。格式为1011 SR PCoffset9.

  14. STR：这是一个基址+偏移量存储指令，用于将寄存器中的值存储到基址+偏移量地址处的内存中。格式为0111 SR BaseR offset6.

  15. TRAP：这是一种软件中断指令，用于调用操作系统服务。格式为 1111 0000 trapvect8.

任务

LC-3 是一种简化的计算机架构，用于教学目的。它有一个小型的指令集，可以执行基本的操作，如加载（LD）、存储（ST）、加法（ADD）、逻辑运算（AND）、跳转（BR）等。

在你的任务中，你需要写一个程序来计算给定数 n 的二进制表示或二进制补码中 0 的数量。这个程序需要满足以下条件：

  1. 如果 n 是奇数，则计算 n 的二进制表示中 0 的数量。
  2. 否则，计算 n 的负数的二进制补码中 0 的数量。
  3. n 的值将手动设置在 x3100 中。
  4. 可以假设 n 是一个范围从 0x0000 到 0x7FFF 的正整数。
  5. 需要计算整个 16 位。
  6. 最终结果应存储在 x3102 中。
