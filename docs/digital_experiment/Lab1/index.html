实验目的与内容

实验目的：学习 verilog 语法，熟悉数电实验的流程
实验内容：本次实验学习了部分简单的 verilog 语法，配置了 vivado-vscode 本地开发环境，编写了 5 段简单的代码，实现了将 8bit 的输入信号按位翻转、用 assign 语句实现两个数取最大值、用 always 和 if-else 语句获得三个数的最大值、通过例化操作获得三个数的最大值、计算出位宽为 3 的信号 in 中为 1 的位的数目、理解Verilog运算符的意义。

逻辑设计

题目 1：将 8bit 的输入信号按位翻转

这里将in的每一位按照从低位到高位的顺序重新排列，然后赋值给了out

赋值情况如下：
$$高\rightarrow 低 = 低 \rightarrow 高$$

[ ] 用于访问向量的单个位或位范围,它允许在不改变原始向量的情况下操作它的部分位

in和out的位顺序并没有本质的区别。在本题中，定义了一个 8 位宽的输入input [7:0] in，那么in[0]就是最低位，in[7]就是最高位，output同理。

代码中的赋值方式为：

$$
assign\quad out = {in[0], in[1], in[2], in[3], in[4], in[5], in[6], in[7]};
$$

它们将按顺序对应地赋给下面的位置

$$
out[7],out[6],out[5],out[4],out[3],out[2],out[1],out[0]
$$

这里将in的每一位按照从低位到高位的顺序重新排列，然后赋值给了out。因此out的位顺序实际上是与输入in相反的。

这就实现了 in 的按位翻转。

选择性必做题 2：Verilog 运算符

题目所给输入：a = 8’b0011_0011, b = 8’b1111_0000。计算结果见注释：

  1. 取非等操作会产生一位的结果，但是我们的输出变量都被定义为八位，输出变量的高位会被接空，自动赋值为0（严格来说应该为 x）
  2. 计算减法时，默认为补码，a - b = a + (-b)，在这一题里表现为将8'b1111_0000取反加一再与8'b0011_0011相加

总结

本次实验中重点学到了以下内容：

  1. 定义的变量（如：out[7:0]）的默认排序为高位 $\rightarrow$ 低位，将输入的变量倒着写再直接赋值给out即可实现按位翻转
  2. 三目条件运算符、if-else语句的使用方式与 C 语言中的类似
  3. 例化接口、变量名尽量有意义
  4. 打表是非常好行为，尽量减少（新手最好不用）for循环
  5. 将少位赋给多位时，最好先将少位的数用1'b0等拼接到与多位数的位数相等，之后再进行加减等操作
  6. 取非等操作会产生一位的结果，但是我们的输出变量都被定义为八位，输出变量的高位会被接空，自动赋值为0（严格来说应该为 x）
  7. 减法运算默认都是补码，无符号，a - b $\rightarrow$ a + (-b)
  8. 在检查排队过程中，听到助教强调：不要进行多次赋值（尤其是count = count + 1），第三次实验才涉及到组合环
