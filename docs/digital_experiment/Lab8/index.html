Lab 8 report

PB22151796 莫环欣

实验目的与内容

本次实验选择了串口方向，完成了串口回显、串口发送、

逻辑设计

T2-1 串口回显

本题完全使用的教程代码，主要目的是熟悉串口操作界面

T2-2 串口发送模块

  • 顶层模块

与文档相比，增加了一个edge_capture三级寄存器边沿检测模块用于检测按钮信号的上升沿

  • 发送模块

补全了状态机

  1. 在Send模块中，当且仅当分频计数器与位计数器同时达到最大值时，将下一个状态改为空闲，避免最后的停止位持续周期不足的情况
  2. 当分频计数器达到最大值时，将其归零，否则自增
  3. 当位计数器达到最大值且分频计数器也达到最大值时，将位计数器其归零，否则在分频计数器达到最大值时自增
    • 避免停止位持续时间过短
  4. 对于输出dout，将位计数器打表，依照其值输出对应位置的内容
    • 同时为了使位计数器与输出内容同时变化，增加分频计数器控制
      • 当分频计数器达到最大值时，提前输出下一位的内容
      • 其它时刻都输出本位置的内容
  • 三级寄存器上升沿检测

正常时刻三个寄存器的值都为 0，当上升沿到来时，r1变为高电平，pos_edge也变为高电平并传递给外界；其余时刻pos_edge都为低电平

T2_A_1 开关输入
